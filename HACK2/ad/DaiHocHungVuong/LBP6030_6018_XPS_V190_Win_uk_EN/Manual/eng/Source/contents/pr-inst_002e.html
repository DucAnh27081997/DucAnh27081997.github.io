Notes and Warnings Regarding Installation
IMPORTANT
If the [User Account Control] dialog box appears, perform the following operations.
For Administrator users, click [Allow] or [Continue].
For Standard users, enter the required Administrator password → click [OK].
If the [Windows Security] dialog box appears, click [Install this driver software anyway].
If you are using Windows 8/8.1/Server 2012/Server 2012 R2, move to the desktop before installing.
NOTE
The 32-bit version of Windows 7 is used for the examples shown in this guide. Depending on the device, driver version, and operating system you are using, displays and procedures may differ from those described in this guide.
