Custom Installation
This section describes installation procedures other than standard installations.
For TCP/IP network environments or USB connection environments, refer to "Standard Installation" to install the driver. If you cannot install the driver by following the "Standard Installation" procedure, or if you need to independently configure settings, refer to the following procedures.
Port Setting Installation
For a device connected to a network or locally, set the port to be used, then install the driver.
Changing a Standard TCP/IP Port
You can change the printer port used by a device connected to a network to a standard TCP/IP port after installing the driver.
Shared Printer Settings
Explains the setting procedure for sharing a printer with computers on the same network.
Installing Using the "inf" File
Install the driver from the [Devices and Printers] (or [Printers and Faxes] or [Printers]) folder of the Windows Control Panel, specifying the "inf" file.
